MobileKit is a small WordPress plugin created to make the development of WP mobile themes and apps simple.

== Description ==
MobileKit is a small WordPress plugin created to make the development of WP mobile themes and apps simple,
allowing web / theme designers to create more immersive user experiences across platforms and devices. (BETA RELEASE)


== Installation ==

1. Upload `mobileKit` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

*Now that you have got MobileKit Installed, you can start creating or using mobile themes

Go to setting (MobileKit Panel) to set up how devices are handled by your themes
Go to UI Options to setup the look and feel of your themes of your theme
Add transitions (Jquery Mobile) to your post and pages from the MobileKit Options meta box
Download the MobileKit Theme to get you started
Visit the MobileKit blog for tips and tutorials on creating themes and apps.
Check out the forum for support or purchase our paid support option
